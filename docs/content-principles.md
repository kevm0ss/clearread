# Content Principles

## The Core Rule

> **"Content should never be reduced. Only reorganised."**

This is the non-negotiable principle of the entire product. It applies to:
- The demo content on the homepage
- The AI output from the URL reformatter
- The system prompts generated by the prompt builder
- The copy on the site itself

---

## What This Means in Practice

**Allowed:**
- Shorter sentences (split one sentence into two)
- Bullet points instead of prose
- Bold key terms
- Reordered information (key point first)
- Added visual structure (headers, numbered steps)
- Defined jargon in brackets
- Summary added at the top

**Not allowed:**
- Removing facts
- Simplifying vocabulary to the point of losing meaning
- Leaving out nuance, caveats, or detail
- "Dumbing down"

---

## Demo Content Rules

The demo on the homepage shows the same document reformatted for each of the four profiles.

**Every fact in the original must appear in every profile version.**

This was learned the hard way during development — early demo versions of the phonological and working memory profiles were missing:
- Marketing strategy as a named section
- The 3-year timeframe for financial forecasts
- The risk identification purpose of the planning process
- The credibility/plan quality point for investors
- P&L statements, cash flow forecasts, and balance sheets as specific outputs
- The executive summary being the only part many investors read

Check every demo version against the original when making changes.

## The Generated Prompt

Every profile prompt must include:

1. A clear profile statement
2. The core rule: "Never simplify content. Only change how it is structured and presented."
3. Profile-specific guidance
4. Numbered formatting rules

The core rule must always be line 2. It is the product's promise expressed as an instruction.

---

## Copy Tone

- Direct and clear
- Short sentences
- No jargon without explanation
- No preamble — lead with the point
- Warm but not patronising
- Never implies the user is less capable

**Wrong:** "This tool helps people who struggle with reading to understand complex information."
**Right:** "Built for the way your brain reads."

---

## Key Copy Lines (Approved)

These have been agreed and should not be changed without discussion:

- **Tagline:** "Content should never be reduced. Only reorganised."
- **Hero stat:** "Dyslexia affects 1 in 10 people."
- **Hero line 2:** "Most AI tools default to dense, hard-to-read text."
- **Hero line 3:** "This changes that. Build your profile. Get your prompt. Done."
- **Preferences instruction:** "Toggle on the formatting rules that help you most. Turn off anything that isn't relevant. Test what works best for you."
- **Prompt warning:** "If you have other instructions to add, keep the total under 1,500 characters."

---

## Prompt Character Limits

| AI Tool | System prompt limit |
|---|---|
| ChatGPT Custom Instructions | ~1,500 characters per field |
| Claude | ~1,500 characters (practical limit) |
| Gemini | Varies |
| Copilot | Varies |

The generated prompt should be well under 1,500 characters with default settings. If a user turns on many optional traits, warn them to check the character count.
